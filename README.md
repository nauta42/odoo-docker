# Dockerizing Odoo

Docker swarm odoo deploy .

## Links

- https://hub.docker.com/_/odoo

## Docker Version

- `docker-compose.yml`: 3.7
- `docker`: 18.+

## Getting Started

```
make init up
```

## Commands

`make run` runs the application

`make stop` stops the application

`make show` shows the current services

`make logs` shows logs
